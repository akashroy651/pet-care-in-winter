# Pet Care in Winter

## Purpose

This project is a simple web app for pet care services during winter.

## Live URL

https://pet-care-in-winter-1.netlify.app/

## Key Features

- View list of pet care services
- Service details page
- Login with Google
- User profile page with update option
- Responsive design

## npm Packages Used

- react
- react-router-dom
- firebase
- tailwindcss
- daisy-ui
